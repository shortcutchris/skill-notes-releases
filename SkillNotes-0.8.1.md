# Skill Notes 0.8.1

The Tasks view now stays calm and predictable when All Notes is resized.
Status, grouping, sorting, workspace, tag, favorite, source-note, and new-task
controls keep their native widths in three clear rows instead of shrinking or
colliding.

All Notes stops at a practical minimum width while keeping the familiar native
sidebar width. Existing workspace drag-and-drop and the rest of the library
behavior remain unchanged.

The bundled CLI and MCP helpers also report version 0.8.1, keeping local agent
connections compatible with this app release.

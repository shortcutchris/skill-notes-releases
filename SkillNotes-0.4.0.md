# Skill Notes 0.4.0

This release adds fast, native checklists to every sticky note.

Type `x` followed by Space at the beginning of a line and Skill Notes turns it into a clickable task. Return continues the checklist with the same indentation. Return on an empty task ends the list, so ordinary writing remains one keystroke away.

Open and completed tasks use accessible, theme-aware native controls, and completed text is struck through. Creation, continuation, termination, and toggling each remain one native Undo action.

Underneath the polished editor presentation, every task remains transparent Markdown (`- [ ]` or `- [x]`). Notes therefore stay portable through revisions, exports, and other Markdown tools without introducing a private rich-text format.

Compatibility: macOS 14 or later.

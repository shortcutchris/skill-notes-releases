# Skill Notes 0.5.1

The attachment shelf is now calmer when closed and more capable when open.

The compact state shows only the attachment count and disclosure—never a file name. In the detailed shelf, use the mouse wheel or trackpad to move horizontally, or use the dedicated previous and next buttons.

Select a file and press Space for native macOS Quick Look. Command-click selects several files. The visible Remove action, Delete, or Backspace removes the complete selection in one operation.

General Settings now controls drag-out behavior:

- Keep the file in its note after copying it elsewhere.
- Remove the note-owned copy only after Finder or another app successfully accepts the drop.

Cancelling a drag never removes an attachment.

All attachments remain local, note-scoped files.

Compatibility: macOS 14 or later.

# Skill Notes 0.4.2

This maintenance release aligns workspace badges with the notes that are actually available in each workspace.

Notes in the Trash no longer inflate the workspace count. Their saved memberships and window placements remain preserved, so restoring a note can still return it to its original workspace.

The workspace detail list, sidebar badge, and VoiceOver value now report the same available-note count.

No note content is changed or deleted by this update.

Compatibility: macOS 14 or later.

# Skill Notes 0.8.10

This release makes tags fully manageable from All Notes.

An active note row can now open its tag editor directly from the context menu,
without opening or moving the sticky note. Every tag in the sidebar has a
discoverable action menu for renaming or removing it globally.

Renaming updates active notes and notes in Trash, so restoring an older note
cannot bring back an obsolete tag. If the new name already exists, Skill Notes
merges the assignments cleanly. Removal shows the number of affected notes and
removes only the tag; note content remains untouched.

## Deutsch

Dieses Release macht Tags vollständig in „Alle Notizen“ verwaltbar.

Über das Kontextmenü einer aktiven Notiz lässt sich der Tag-Editor jetzt direkt
öffnen, ohne die Haftnotiz zu öffnen oder zu verschieben. Jeder Tag in der
Seitenleiste besitzt ein sichtbares Aktionsmenü zum globalen Umbenennen oder
Entfernen.

Das Umbenennen aktualisiert aktive Notizen und Notizen im Papierkorb, damit
beim Wiederherstellen kein veralteter Tag zurückkehrt. Existiert der neue Name
bereits, führt Skill Notes die Zuordnungen sauber zusammen. Vor dem Entfernen
wird die Anzahl der betroffenen Notizen angezeigt; der Inhalt der Notizen
bleibt unangetastet.

# Skill Notes Releases

This public repository contains signed and notarized Skill Notes release archives and the stable Sparkle update feed.

- Download releases from [GitHub Releases](https://github.com/shortcutchris/skill-notes-releases/releases).
- Sparkle reads [`appcast.xml`](https://raw.githubusercontent.com/shortcutchris/skill-notes-releases/main/appcast.xml) and verifies every archive with the public key embedded in Skill Notes.
- Application source, Apple credentials, notarization credentials, and the private Sparkle signing key are not stored here.

The application source remains in a separate private development repository.

# Skill Notes 0.6.0

Attachments are now available directly in All Notes and every workspace, favorite, tag, or search result.

Each note row uses the same calm, filename-free compact shelf as the sticky note. Click the attachment count to reveal native file cards. From there you can:

- open a file or reveal it in Finder;
- select several files with Command-click;
- press Space for macOS Quick Look;
- scroll horizontally with a mouse or trackpad, or use the page buttons;
- remove the selection with the visible action, Delete, or Backspace;
- drag files into Finder, Mail, or another accepting app.

Drop one or more files anywhere on an active note row to copy them into that note, including a note without existing attachments. Trashed notes keep their existing files accessible but do not accept new imports.

The General Settings drag-out choice applies everywhere: keep the note-owned copy, or remove it only after an external destination successfully accepts the drop. Cancelling a drag never removes a file.

All storage remains local and note-scoped in this release.

Compatibility: macOS 14 or later.

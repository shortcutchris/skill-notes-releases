# Skill Notes 0.1.2

This update makes the sticky-note pin behave like a real window pin and repairs the direct route from AI readiness to configuration.

Pinning a note now immediately moves that note to the floating macOS window level so it remains above ordinary application windows. Unpinning returns it to the normal window level, allowing another app or the All Notes window to cover it. The choice remains stored per note and continues to determine which notes appear in Favorites.

The AI readiness popover's AI Settings button now opens Settings reliably and selects the AI pane directly. The pin also exposes clearer English and German help and accessibility state text.

This release does not change AI request scope or privacy behavior. Note content remains local unless the user explicitly invokes an AI skill.

Compatibility: macOS 14 or later.

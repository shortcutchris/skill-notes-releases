# Skill Notes 0.5.0

Attachments now live in a dedicated shelf at the bottom of each sticky note.

The compact shelf stays calm: it shows the first file's type and name plus the number of remaining attachments. Hover only highlights it. Details open only after an explicit click.

The opened shelf adds native file previews, type and size metadata, Command-click multiple selection, double-click Open, contextual file actions, and copy-only drag-out to Finder, Mail, or another compatible app.

You can also drop several files anywhere on a note or choose several files from the attachment picker. A deliberate dragover opens the shelf after a short delay; cancelling restores its previous state, while a successful drop keeps it open to confirm the imported files.

All attachments remain local, note-scoped files.

Revision history ordering is also stable when several changes happen within the same millisecond.

Compatibility: macOS 14 or later.

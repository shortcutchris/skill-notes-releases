# Skill Notes 0.8.2

Skill Notes can now turn an image on the system clipboard into a normal note
attachment. Focus a sticky note, press Command-V, and the image appears in the
existing attachment shelf without adding hidden markup to the note text.

The app briefly confirms the import and offers Undo. Common clipboard image
formats and image-file URLs are supported, while ordinary text paste behaves
exactly as before.

## Deutsch

Skill Notes kann ein Bild aus der Systemzwischenablage jetzt als normalen
Notizanhang übernehmen. Einfach eine Haftnotiz fokussieren und Befehl-V drücken:
Das Bild erscheint in der vorhandenen Anhangsablage, ohne unsichtbare
Markierungen in den Notiztext einzufügen.

Die App bestätigt den Import kurz und bietet Rückgängig an. Verbreitete
Bildformate und Bilddatei-URLs werden unterstützt; normales Einfügen von Text
funktioniert unverändert.
